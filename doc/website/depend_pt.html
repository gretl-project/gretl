Dependências de gretl

Gretl depende de várias bibliotecas e do programa gnuplot, para algumas das suas funcionalidades. A tabela abaixo mostra um resumo destas dependências. A ligação à esquerda leva à página principal do pacote em questão (válido a partir de Abril de 2002), enquanto a ligação à direita leva à pesquisa do pacote em rpmfind.net.

Pacote Relação com gretl Funcionalidade rpmfind
gtk Obrigatório Fornece a interface gráfica do gretl Procurar
gtkextra Facultativo Diálogo de selecção de ficheiros, vista da sessão em modo ícone, folha de cálculo de edição de dados Procurar
libxml Obrigatório Abre e grava os ficheiros no formato predefinido Procurar
lapack Obrigatório Várias funções de álgebra linear Procurar
gnuplot Obrigatório Gera os gráficos Procurar
gdk-pixbuf Obrigatório Gestor de imagens em formato PNG; torna os gráficos gnuplot interactivos
 Procurar
readline Facultativo Fornece um agradável editor de linha de comandos em gretlcli, a versão em modo de linha de comandos
 Procurar
gnome Facultativo gretl fica integrado no ambiente de trabalho gnome (ícone e associação de ficheiros, sistema de ajuda  HTML e outras coisas) Procurar
libole2 Facultativo [1] Importa dados do MS Excel Procurar
GMP Facultativo Suporte para o plugin OLS de precisão múltipla Procurar

Note

1. A parte relevante do código de libole2 está integrada no pacote de código fonte do gretl, e se não existir no seu sistema este código será compilado dentro do gretl.
