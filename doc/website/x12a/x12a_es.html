gretl y X-12-ARIMA

X-12-ARIMA es el programa que usa actualmente el U.S. Census Bureau para corrección estacional. La página web de este programa está en www.census.gov. Por favor, consulte esa página para ver su documentación y otros detalles.

Si Vd. desea instalar X-12-ARIMA para utilizarlo junto con gretl bajo GNU/Linux puede

  • Compilar el código fuente Vd. mismo, en cuyo caso le sugiero que ejecute el guión de shell x12arima.sh. Este se suministra en el paquete de fuentes de gretl; Vd. también puede verlo o descargarlo desde aquí. Este guión descarga los archivos necesarios, hace unas ligeras modificaciones que se necesitan para compilarlo bajo Linux, compila el programa (para esto, necesita Vd. el compilador GNU Fortran, g77), y lo instala en el lugar que Vd. elija.

  • Instalar un paquete binario RPM, x12a-0.2.10-1.i386.rpm (Este paquete fue creado bajo Red Hat Linux 7.3).

Si desea usar X-12-ARIMA junto con gretl bajo Windows, debería descargar el autoinstalador, x12a_install.exe y ejecutarlo. Se le preguntará por un directorio de instalación, por defecto c:\userdata\x12arima. Este paquete contiene el programa x12a.exe adecuado para una consola de win32 (Ventana MSDOS en Windows), algunos ejemplos y documentación en formato PDF. Deseo dar las gracias a Brian Monsell, de la Statistical Research Division of the U.S. Census Bureau, por su permiso para redistribuir X-12-ARIMA en esta forma.

La interconexión con X-12-ARIMA es nueva en la versión 1.0.2 de gretl y por ahora es bastante limitada, pero si hay interés en esta opción probablemente procederé a desarrollarla con más detalle. Las siguientes imágenes ilustran lo que actualmente se puede hacer.

Después de abrir un conjunto de datos adecuado (p.ej. que contenga series temporales estacionales), seleccione una variable en la ventana principal de gretl y abra el menú de 'Variable'. Seleccione ahora "Análisis X-12-ARIMA".

Entonces Vd. verá un cuadro de diálogo que le da la oportunidad de guardar (en el conjunto de datos actual de gretl) hasta tres de las series generadas por X-12-ARIMA, concretamente, la versión corregida de estacionalidad de la serie seleccionada, el componente tendencia/ciclo estimado de la serie y el componente "irregular" estimado. En este momento Vd. también puede elegir si desea o no obtener un gráfico que muestre estas series.
