Gretl: Gnu Regression, Econometrics and Time-series Library

for Mac OS X

First you need to determine if your Mac has an Intel or a PPC processor. Newer Macs (since 2006) all run Intel. If you have an older Mac and are not sure, click on the Apple icon at the top left of your screen and select "About this Mac". In the window that appears you'll find the OS X version number and also the processor type.

If you have an intel Mac running OS X 10.6 (Snow Leopard) or higher. we recommend the "quartz" download below. Some users may prefer to use the X11 version of gretl.

When you're ready, select one of the links below.

  • Intel Mac, quartz
  • Intel Mac, X11
  • PPC Mac (old)
