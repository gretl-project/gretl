Gretl: Gnu Regression, Econometrics and Time-series Library

Versione MS Windows

Download

A partire dalla versione 1.9.4, gretl richiede Windows XP o successivo, e un processore che supporta il set di istruzioni SSE2. Quest'ultimo è disponibile in tutti i processori moderni, ma manca nei processori AMD precedenti all'Athlon 64 e nei processori Intel precedenti al Pentium 4. Se si cerca una versione di gretl che funzioni su vecchi sistemi, si veda sotto.

Download

Se si hanno i permessi di amministrazione su Windows, scegliere un pacchetto autoinstallante nella prima colonna della tabella seguente: una volta scaricato basta eseguire il file exe. Questa soluzione in genere è adatta per la maggior parte degli utenti.

Se non si hanno i permessi di amministrazione su Windows, bisogna scaricare il pacchetto zip e scompattarlo in una cartella su cui si ha il permesso di scrittura. Nota: occorre conservare la struttura di cartelle dell'archivio, altrimenti gretl non funzionerà. Nei programmi per scompattare i file, questa opzione si chiama talvolta "abilita cartelle". L'archivio è in una cartella chiamata gretl. Ad esempio, se si scompatta l'archivio in una cartella chiamata c:\userdata, il file eseguibile di gretl sarà in c:\userdata\gretl\gretl.exe.

Lo "snapshot" attuale di gretl è più aggiornato: spesso contiene correzioni a bug, ma talvolta contiene nuovi bug. Per vedere le novità contenute nello snapshot, si può vedere il Change log di gretl (la voce "in progress").

gretl per vecchi computer

Per sistemi meno recenti, gretl 1.9.3 gira su Windows 2000 (e non richiede SSE2), mentre gretl 1.6.5 è l'ultima versione che gira su Windows 98.

Supporto multilingua

Oltre alla lingua inglese, gretl supporta il francese, l'italiano, lo spagnolo, il tedesco, il basco, il portoghese, il polacco, il turco, il ceco, l'albanese, il russo e il cinese tradizionale. Di solito non occorre fare nulla di speciale per poter usare gretl nella lingua locale: il programma dovrebbe usare automaticamente le traduzioni disponibili basandosi sull'impostazione della lingua attiva sul computer. In alcune circostanze, però, potrebbe essere necessario modificare l'impostazione della lingua sul computer, usando il pannello di controllo di Windows.

Aggiornare gretl

Prima di aggiornare gretl è consigliabile eseguire il programma di disinstallazione unins000.exe, che si trova nella cartella principale di gretl. In questo modo si eviteranno possibili problemi dovuti a "vecchie" DLL o impostazioni del registro di Windows. Attenzione: disinstallando il programma non verranno rimossi eventuali file creati nella cartella gretl/user, ma verranno cancellate le personalizzazioni fatte nel menù "Preferenze", che quindi dovranno essere reimpostate una volta avviata la nuova versione del programma.

Eseguire gretl via rete

Sono disponibili istruzioni per configurare gretl per Windows in modo da essere eseguito via rete, ad esempio in un laboratorio informatico.
