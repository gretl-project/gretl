Gnu Regression, Econometrics and Time-series Library

Es un paquete de software para análisis econométrico escrito en el lenguaje de programación C. Es software libre y de código abierto. Vd. puede redistribuirlo y/o modificarlo bajo los términos de la Licencia Pública General GNU(GPL) según se ha especificado por la Free Software Foundation.

Conferencia en 2009

La primera conferencia sobre gretl tuvo lugar en la Universidad del País Vasco, Bilbao, España, los días 28 y 29 de mayo de 2009: más información. Muchas gracias a los organizadores, Ignacio Díaz-Emparanza, Petr Mariel y María Victoria esteban, por organizar un evento tan productivo y tan entretenido. Características

  • Interface fácil e intuitiva (ahora en Alemán, Checo, Español, Francés, Italiano, Polaco, Portugués, Ruso, Turco y Vasco además del Inglés)

  • Incluye una gran variedad de estimadores: mínimos cuadrados, máxima verosimilitud, GMM; de una sola ecuación y de sistemas de ecuaciones

  • Métodos de series temporales: ARMA, GARCH, VARs y VECMs, contrastes de raíces unitarias y de cointegración, etc.

  • Los resultados de los modelos se pueden guardar como ficheros LaTeX, en formato tabular y/o de ecuación.

  • Incluye un lenguaje de programación vía 'scripts' (guiones de instrucciones): las órdenes se pueden introducir por medio de los menús o por medio de guiones.

  • Estructura de bucles de instrucciones para simulaciones de Monte Carlo y procedimientos de estimación iterativos.

  • Controlador gráfico mediante menús, para el ajuste fino de los gráficos Gnuplot

  • Enlace a GNU R para análisis más sofisticados de los datos

Formatos de datos

Lee sus propios ficheros de datos en formato XML, ficheros CSV (Comma separated values), los de las hojas de cálculo Excel, Gnumeric y Open Document, ficheros .dta de Stata, ficheros .sav de SPSS; ficheros de trabajo de Eviews, ficheros de datos de JMulTi; su propio formato binario de bases de datos (que permite frecuencias mixtas de los datos y larguras mixtas de las series), bases de datos de RATS4 y de PC-Give. Incluye una base de datos macroeconómicos de los EEUU. Ver también la página datos para gretl.

Gretl llama al programa 'gnuplot' para generar los gráficos. La última versión oficial de gnuplot es la 4.2.6 (fechada en septiembre de 2009), que está disponible en gnuplot.sourceforge.net.La versión actual de desarrollo de gnuplot es la 4.5. Esta versión incluye varias características nuevas que son convenientes para su uso con gretl, y gnuplot 4.5 se suministra incluído en el paquete de gretl para MS Windows. (El paquete para OS X incluye gnuplot 4.3, que también contiene características convenientes para su uso con gretl.)

Manual

El manual de Gretl viene en dos ficheros PDF, la Guía del Usuario y la Guía de Instrucciones. La versión inglesa de ambas, formateada para el tipo de papel letter de EE.UU., está incluída en el paquete fuente de Gretl y en las distribuciones binarias. Gretl accederá a las otras versiones disponibles --es decir, la versión en ingles en papel a4, y la versión en Italiano-- bajo demanda, via Internet. Los ficheros de manual también están disponibles aquí.

Si Vd. no tiene el software para leer ficheros PDF puede descargarlo de forma gratuita desde el sitio Web de Adobe.

Usa Vd. gretl?

Si lo hace, es posible que desee unirse a la lista de correo gretl-users. Es una lista de moderado volumen de tráfico donde los usuarios piden y ofrecen ayuda, discuten sobre nuevas características, etc. Es posible buscar información en los archivos de la lista. También existe la lista gretl-announce; Esta es una lista de sólo lectura, que envía avisos cuando se publican nuevas versiones de gretl.

Vd. puede enviar informes de error y peticiones de nuevas utilidades por medio del sistema de seguimiento de SourceForge. Pero por favor: le pedimos que utilice el sistema de seguimiento de Sourceforge para enviar infromes de error sólo si Vd. está realmente seguro de que ha encontrado un error; si Vd. tiene dudas, por favor envíe primero un mensaje a una de las listas de correo de gretl. (No es necesario decir que si gretl se cuelga ¡Vd ya ha encontrado un error seguro!)

Contribuir al desarrollo de Gretl

Tiene Vd. alguna idea o código con el cual le gustaría contribuir al desarrollo de Gretl? Si es así, estaríamos encantados de escucharle. Vd. probablemente debería unirse a la lista de correo gretl-devel (es una lista de bastante poco tráfico).

La única condición respecto al código que se incorpora a Gretl es que Vd. debe estar conforme en que aparezca bajo la Licencia Pública General GNU (GPL), es decir, Vd. debe de estar de acuerdo en que se distribuya el código fuente.

Como se ha mencionado más arriba, Gretl está escrito (principalmente) en el lenguaje de programación C, el lenguaje más común en el mundo del software de código abierto. Sin embargo, no todo el mundo se siente cómodo al escribir en C. Debe entender que esto no es una barrera para contribuir a este proyecto con código o algoritmos. Si Vd. puede proporcionar un algoritmo bien especificado para llevar a cabo alguna interesante tarea econométrica, probablemente no nos será difícil reescribirlo en C o hacer que Gretl lo interprete. Esto incluye código en Fortran o C++; código escrito en algún lenguaje de alto nivel para manipulación de matrices tal como Ox, Gauss, Octave, Matlab o R; e incluso pseudo-código, es decir un algoritmo explicado explícitamente, paso a paso, pero no escrito en ningún lenguaje de programación en concreto.
